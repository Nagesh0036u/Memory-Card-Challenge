# 🧠 Memory Card Challenge

Memory Card Challenge is an interactive memory game developed using **Python** and **Streamlit**. The player is shown a sequence of randomly selected words for a few seconds before they disappear. The objective is to recall and enter as many words as possible from memory. The application automatically evaluates the answers, calculates the score, and stores the results in a persistent leaderboard.

Unlike a simple guessing game, this project demonstrates how state management can be used to build an interactive application that progresses through multiple stages while maintaining user data throughout the session.

---

## Project Overview

The application follows a state-driven workflow that guides the player through each phase of the game.

1. Enter player name
2. Select difficulty level
3. Memorize the displayed words
4. Wait for the countdown to finish
5. Enter remembered words
6. View score
7. Save score to the leaderboard

---

## Features

- Player login system
- Three difficulty levels
- Random unique word generation
- Five-second memory challenge
- Automatic answer validation
- Case-insensitive comparison
- Score calculation
- Persistent leaderboard using CSV
- Clean and responsive Streamlit interface
